# Nordnet Summer School

Introduction to energy system modelling with PyPSA for Nordnet Summer School 2026 in Trondheim, Norway.

## Usage

### Building the book

If you'd like to develop and/or build the Nordnet Summer School book, you should:

1. Clone this repository
2. Run `uv sync`
3. (Optional) Edit the books source files located in the `nordnet/` directory
4. Run `uv run jupyter book start` or `jupyter-book build --html --execute` to build the book

A fully-rendered HTML version of the book will be built in `_build/html/`.

### Hosting the book

Please see the [Jupyter Book documentation](https://jupyterbook.org/publish/web.html) to discover options for deploying a book online.
